# reference files for non-regression testing

this directory contains reference data generated from MATLAB fdnToolbox for comparison with pyFDN implementations.

## generating reference datA


to generate the reference data for one-pole absorption testing:

1. navigate to `fdnToolbox/Examples/`
2. run the MATLAB script: `example_onePoleAbsorption_export.m`
3. this will generate `onePoleAbsorption_reference.mat`
4. vopy the .mat file to this directory

## files

- `onePoleAbsorption_reference.mat`: teference data from MATLAB fdnToolbox one-pole absorption example
  - generated by: `fdnToolbox/Examples/example_onePoleAbsorption_export.m`
  - contains: delays, feedbackMatrix, absorption coefficients (b,a), impulse response, and parameters
  - used by: `tests/test_one_pole_absorption_regression.py`

## test coverage

the regression tests compare:
- one-pole absorption filter coefficients (b, a)
- random orthogonal matrix generation (with same seed)
- basic FDN parameters consistency
- IR characteristics
